# dotfiles

## Requirements
    - `git`
    - `stow`
    - `curl`
    - `tmux`
    - `vim`
    - `neovim`
    - `zsh`

## Installation
```shell
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
git clone https://github.com/tmux-plugins/tpm ~/.config/tmux/plugins/tpm
stow .
```
